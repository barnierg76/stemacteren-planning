# Stemacteren.nl Workshop Planning System - Briefing

> Oorspronkelijke briefing. Alle gegevens hieronder zijn VOORBEELDDATA.
> In het systeem is ALLES configureerbaar via admin interface.

## Ontwerpprincipe

**NIETS IS HARDCODED**

- Workshoptypes: toevoegen/wijzigen/verwijderen via UI
- Locaties: toevoegen/wijzigen/verwijderen via UI
- Teamleden: toevoegen/wijzigen/verwijderen via UI
- Business rules: configureerbaar via settings
- Prijzen, tijden, limieten: allemaal aanpasbaar

Het systeem moet flexibel genoeg zijn om:
- Nieuwe workshoptypes toe te voegen
- Locaties te sluiten of openen
- Regels aan te passen zonder code-wijzigingen
- Tarieven te updaten
- Nieuwe teamleden toe te voegen

## Huidige Situatie (als startdata)

### Locaties (configureerbaar)
- Amsterdam, Utrecht, Leiden
- Elke locatie heeft eigen beschikbare dagen

### Workshoptypes (configureerbaar)
- BWS, BTC, VWS, IWS, AWS, LBWS, etc.
- Elk type heeft eigen regels, prijzen, limieten

### Team (configureerbaar)
- Interne docenten met voorkeuren en limieten
- Externe docenten
- Technici

### Business Rules (configureerbaar)
- Publicatietermijnen
- Overlap-regels
- Energie-regels
- Conversiepercentages
- Targets
